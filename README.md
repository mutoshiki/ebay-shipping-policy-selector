# eBay 配送ポリシー選択ツール

梱包後重量と、送料を除いたeBayの商品価格から、使用する配送ポリシーを判定する静的サイトです。

## 判定ルール

- 重量: 300g / 500g / 800g / 1000g / 1500g / 2000g
- 価格: 100USD以下 / 101〜250USD
- 2000g超または250USD超: 個別確認

ポリシー名は次の形式です。

```text
CAM{重量区分}{価格区分}
```

例:

```text
CAM800G100USD
CAM1500G101250USD
```

## 使い方

`index.html` をブラウザで開くだけで使用できます。外部ライブラリやサーバーは不要です。

## GitHub Pagesへ公開

1. このフォルダの中身をGitHubリポジトリへアップロード
2. リポジトリの Settings → Pages を開く
3. Branch を `main`、フォルダを `/ (root)` に設定
4. Saveを押す

## ファイル

- `index.html`: 画面構造
- `styles.css`: 表示デザイン
- `script.js`: 判定処理
- `favicon.svg`: アイコン
