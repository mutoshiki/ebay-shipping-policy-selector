# Film Camera Shipping Guide

フィルムカメラ、フィルム対応交換レンズ、アクセサリーの重量を検索し、梱包後重量と商品価格からeBay配送ポリシーを判定する静的Webアプリです。GitHub Pagesまたは単体HTMLとして動作し、サーバーは不要です。

## 主な機能

- 商品名、型番、日本語メーカー名、海外での別名から検索
- 参考重量とジャンル推定を明確に区別
- カメラ・レンズ・アクセサリーをメーカー、ジャンル、マウント、重量品質で絞り込み
- 同ジャンルの近い参考機種と重量を比較
- 一覧にない商品をジャンルから見積もり、そのまま配送判定に反映
- ジャンルごとの登録数、参考重量数、一般的な重量範囲、代表機種を表示
- 本体重量、梱包材、ケース、ストラップ、フィルム、元箱などを合算
- 梱包後の実測重量による上書き
- 商品価格と重量から12種類のCAM配送ポリシーを判定
- 米国、EU27、その他対応国、国際基準送料をUSDと日本円で併記
- ドル円レートをブラウザに保存
- 自分の実測値を端末に保存し、JSONで書き出し・読み込み
- CSV・JSON形式のデータを同梱

## データ件数

2026年6月30日生成版は、**1,341件**のフィルム関連商品を収録しています。

- カメラ：560件
- フィルム対応レンズ：738件
- アクセサリー：43件
- 個別の参考重量：416件
- ジャンル：38分類

個別重量が未確認の商品には、同ジャンルの参考重量分布から求めた推定範囲を表示します。デジタル専用マウント、APS-C専用レンズ、ミラーレス専用レンズは可能な範囲で除外しています。

## 重量データの品質表示

- **B 参考重量**：メーカー資料、取扱説明書、公式アーカイブ等を中心に整理した代表値
- **C 参考重量**：公開された仕様表や複数の二次資料で広く使われる代表値。地域仕様や付属品差に注意
- **D ジャンル推定**：個別重量が未収録のため、同ジャンルの参考機種から範囲を算出
- **端末の実測**：利用者がこのブラウザに保存した値。梱包条件が同じなら最優先

## 重要な注意

重量は同一機種でも、地域仕様、データバック、ファインダー、フィルムバック、電池、ストラップ、キャップ、レンズなどで変わります。中判カメラは「ボディのみ」と「標準構成」で特に大きな差が出ます。

本アプリの重量は、商品候補と配送区分を事前に絞るための情報です。最終的な出品・発送には、付属品と梱包材をすべて含めた梱包後の実測値を使用してください。

## データ更新

`data/camera-weight-database.csv` は確認・編集用です。アプリが直接読み込むデータは `data/database.js` です。大量更新は `build_film_camera_app.py` 内の参考データと分類規則を編集し、再実行してください。

## 主な基礎データ

- 公開モデルカタログ：`w84thesun/film-camera-database-sample`
- 現行アナログカメラの分類参考：`analog-photography-space/analog-camera-space`
- 個別参考重量：メーカー公称仕様、取扱説明書、公式アーカイブ、公開されたカメラ資料を照合した代表値

公開運用や再配布では、各情報源の利用条件を確認してください。
